🔥 Lấy 3 giá trị còn thiếu từ Firebase

Bước 1 — Mở Firebase Console

Click nút bên dưới để mở đúng trang cần lấy config:

console.firebase.google.com
→ Project: giaohangdn
→ ⚙️ Project Settings (bánh răng góc trái)
→ Tab "Your apps" → Web app (biểu tượng </>)
→ Kéo xuống phần SDK setup and configuration
→ Chọn "Config" (radio button)

Bước 2 — Dán 3 giá trị vào đây

Copy từng giá trị trong khung config Firebase rồi dán vào ô bên dưới:

Bắt đầu bằng "AIzaSy", dài ~39 ký tự

Dãy số, ~12 chữ số

Bắt đầu bằng "1:", có dấu ":"

Bước 3 — Lưu vào app

Sau khi điền đủ 3 giá trị, click nút bên dưới để ghi vào app.js:
