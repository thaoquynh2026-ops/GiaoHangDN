Kết nối Firebase – Hướng dẫn từng bước cho người mới

⚫ Chưa kết nối Firebase
1
Tạo tài khoản Firebase
Vào website Firebase, đăng nhập Gmail
1
Bấm nút bên dưới để mở Firebase Console (sẽ mở trong trình duyệt)

2
Trang hiện ra → bấm nút "Sign in" (góc trên phải) → chọn tài khoản Gmail của bạn
3
Sau khi đăng nhập, bạn thấy trang chủ Firebase Console. Bấm nút bên dưới khi xong để sang bước tiếp:

2
Tạo Project mới
Tên project: giaohangdn
1
Ở trang Firebase Console → bấm "Add project" (hoặc "Create a project")
2
Ô "Project name" → gõ: giaohangdn → bấm Continue
3
Trang tiếp theo có "Google Analytics" → gạt sang OFF (tắt đi cho đơn giản) → bấm "Create project"
4
Chờ ~20 giây cho đến khi thấy dấu ✅ và chữ "Your new project is ready" → bấm Continue

3
Tạo Realtime Database
Đây là nơi lưu đơn hàng online
1
Ở menu bên trái → bấm "Build" → chọn "Realtime Database"
📁 Build
  ├── Realtime Database ← bấm vào đây
  ├── Firestore Database
  ├── Storage
  └── Hosting
2
Bấm nút "Create Database" (màu xanh dương)
3
Hộp thoại hiện ra:
• "Database location" → chọn Singapore (asia-southeast1) (gần Việt Nam nhất)
• Bấm Next
4
Trang tiếp theo → chọn "Start in test mode" (cho phép đọc/ghi tự do)
● Start in test mode ← chọn cái này
○ Start in locked mode
→ bấm "Enable"

4
Đăng ký Web App & lấy Config
Lấy "chìa khóa" kết nối vào app
1
Bấm biểu tượng ⚙️ cạnh chữ "Project Overview" (góc trên bên trái) → chọn "Project settings"
⚙️ Project settings ← bấm vào đây
Usage and billing
Users and permissions
2
Kéo xuống phần "Your apps" → bấm biểu tượng </> (Web)
Your apps
</> iOS Android Unity Flutter
3
Ô "App nickname" → gõ giaohangdn-web → bấm "Register app"
4
Trang tiếp theo sẽ hiện đoạn code. Bạn sẽ thấy dạng như thế này:
const firebaseConfig = {
  apiKey: "AIzaSy...",
  authDomain: "giaohangdn.firebaseapp.com",
  databaseURL: "https://giaohangdn...firebasedatabase.app",
  projectId: "giaohangdn",
  storageBucket: "giaohangdn.appspot.com",
  messagingSenderId: "123456789",
  appId: "1:123..."
};
⚠️ Quan trọng: Phải có dòng databaseURL. Nếu không thấy dòng này → quay lại Bước 3 tạo Realtime Database trước!
→ Bấm Continue to console

5
Dán Config vào app.js
Điền từng giá trị vào ô bên dưới
1
Nhìn vào đoạn config Firebase bạn vừa lấy → điền từng giá trị vào các ô bên dưới:
📋 Xem trước — sẽ được tự động ghi vào app.js:
← Điền các ô trên để xem preview
2
Khi đã điền xong tất cả → bấm nút bên dưới để tự động ghi vào app.js:

 
✅ Đã lưu thành công! Config đã được ghi vào app.js
Nếu nút Lưu không hoạt động: bấm Sao chép config → mở file app.js bằng Notepad → tìm dòng const FIREBASE_CONFIG → xóa cả khối và dán đoạn vừa copy vào.

6
Mở quyền Database Rules
Cho phép app đọc & ghi data
1
Quay lại Firebase Console → Build → Realtime Database → tab "Rules"
Data   Rules   Backups   Usage
─────────────────────
2
Xóa nội dung cũ trong ô Rules → dán đoạn code này vào: 
{
  "rules": {
    ".read":  true,
    ".write": true
  }
}
3
Bấm nút "Publish" (màu xanh) để lưu Rules

7
Kiểm tra kết nối
Mở app và thử đặt đơn hàng
1
Bấm nút bên dưới để mở app GiaoHangDN:

2
Nhìn vào góc trên header — nếu thấy badge 🟢 Online → Firebase đã kết nối!
🛵 GiaoHangDN   🟢 Online ← phải thấy badge này
3
Thử đặt 1 đơn hàng bất kỳ → quay lại Firebase Console → Realtime Database → Data → Nếu thấy data xuất hiện là thành công 100%!
