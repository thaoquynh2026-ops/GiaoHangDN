🛵 Kế hoạch kiếm tiền với GiaoHangDN

Từ app → tiền thật trong 30 ngày

1

Kết nối Firebase (đồng bộ mọi thiết bị)

Hiện tại app chỉ lưu data trên 1 máy. Firebase giúp khách đặt trên điện thoại, tài xế thấy ngay trên máy mình.

Làm theo 5 bước (miễn phí):
  1. Vào console.firebase.google.com → đăng nhập Google
  2. Bấm "Add project" → đặt tên giaohangdn
  3. Vào Build → Realtime Database → Create database → Start in test mode
  4. Vào Project Settings ⚙️ → Your apps → Web → copy đoạn firebaseConfig
  5. Mở file app.js → dán config vào FIREBASE_CONFIG = { ... }
Miễn phí 30 phút
2

Deploy lên internet (có link chia sẻ)

Sau khi deploy, bạn có URL như giaohangdn.web.app để gửi cho khách hàng.

Dùng Firebase Hosting (miễn phí):
  1. Mở terminal → gõ: npm install -g firebase-tools
  2. firebase login (đăng nhập Google)
  3. Vào folder GiaoHangDN: cd Desktop\GiaoHangDN
  4. firebase init hosting → chọn project → public directory: . → Single-page: No
  5. firebase deploy → lấy URL
Miễn phí 1 giờ
3

Tuyển 3–5 tài xế đầu tiên

Đây là bước quan trọng nhất. Không có tài xế = không giao được đơn.

Ngày 1–3: Đăng lên các group Facebook Đà Nẵng: "Tuyển shipper bán thời gian, chạy theo đơn, 15% hoa hồng"
Ngày 4–5: Lập group Zalo với tài xế → gửi link driver.html để họ nhận đơn
Ngày 6: Chạy thử 1 đơn giả để tài xế quen luồng
Tài xế nhận được: 15% mỗi đơn. Đơn 100k → tài xế được 15k.
Bạn giữ: 85% – chi phí quán → lãi ~5–10% mỗi đơn.
4

Ký kết với 5–10 quán ăn gần nhà

Đến trực tiếp, nói đơn giản: "Em có app giao hàng, quán chỉ cần nấu, em lo hết phần giao. Quán không mất gì, chỉ trả 10% khi có đơn."

  • Ưu tiên quán chưa có ShopeeFood/GrabFood
  • Quán bình dân, đông khách sinh viên
  • Khu vực: Hải Châu, Thanh Khê, Liên Chiểu
Cập nhật quán vào app: Mở app.js → tìm RESTAURANTS → thêm quán thật vào mảng đó.
5

Marketing 0 đồng để có đơn đầu tiên

  • 📱 Đăng lên Facebook groups Đà Nẵng: "Giao hàng nhanh, phí chỉ 15k"
  • 💬 Gửi link app vào group Zalo phòng trọ, khu dân cư
  • 🎁 Tuần đầu miễn phí giao hàng để có đánh giá đầu tiên
  • 📸 Chụp ảnh đơn hàng thật → đăng Facebook Reels
  • 🤝 Nhờ quán ăn chia sẻ link app cho khách của họ
