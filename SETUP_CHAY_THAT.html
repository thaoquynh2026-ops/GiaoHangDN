📱 Bước 1 — Telegram nhận đơn Bắt buộc

Khi có đơn mới, tin nhắn sẽ gửi thẳng vào điện thoại mày.

Chưa có bot? Làm theo 3 bước:
1. Mở Telegram → tìm @BotFather → gửi /newbot
2. Đặt tên → BotFather gửi lại Token (dạng 123...:AAF...) → copy vào ô trên
3. Tìm @userinfobot → gửi bất kỳ tin → nó trả về Id → copy vào Chat ID

🔥 Bước 3 — Firebase Tuỳ chọn

Không có Firebase vẫn chạy được. Cần Firebase khi dùng nhiều thiết bị cùng lúc (admin + tài xế cùng xem đơn real-time).

Lấy tại: console.firebase.google.com → project giaohangdn → ⚙️ Project Settings → Your apps → Config
